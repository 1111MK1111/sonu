# Version History

## Version 2.0.0 (Current)
- Light glassy white Apple-style theme
- SONU branding
- Sidebar navigation (Home, Dictionary, Snippets, Style, Notes, Settings)
- Settings page with modals
- Live transcription preview
- Inline history editing
- Stats tracking (streak, words, WPM)
- Fixed hotkey capture to prevent triggering dictation during setup

## Version 1.0.0 (Previous)
- Dark theme with purple accents
- Simple layout with header, output, and controls
- Settings inline in main window
- "Flow" branding
- Basic history display
- Toggle and hold recording modes
